# Flutter Instagram Clone

An Instagram clone made with Flutter and Firebase Auth, Firebase Storage, Firebase Firestore for the backend

## To start development

After cloning this repository, migrate to ```flutter-instagram-clone-main``` folder. Then, follow the following steps:

- Create Firebase Project
- Enable Authentication
- Make Firestore Rules
- Create Android, iOS & Web Apps
- Take Web FirebaseOptions and put it in main function in main.dart file replacing my keys (My keys wont work as I deleted my project)
Then run the following commands to run your app:
```bash
  flutter pub get
  open -a simulator (to get iOS Simulator)
  flutter run
  flutter run -d chrome --web-renderer html (to see the best output)
```
